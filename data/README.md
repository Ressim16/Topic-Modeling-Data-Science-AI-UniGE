# Data Directory

Due to size constraints and professional secret, data files are not included in this repository.

## How to use the project with data

1. Have a dataset with the required structure (as described in the main notebook)
2. Run the data collection cells in the main notebook
3. Or ask me for the data files directly
4. Create a directory named `data` in the root of this repository
5. Place files in this directory, ensuring they are named as follows or changing the paths in the code accordingly:
   - cleaned_combined_dataset.csv
   - potential_duplicates.csv
   - pubmed_5000.csv
   - scopus_5000.csv
